Why It Doesn't Work

This variable...

Is local to the function component.

So, any part of react(like onClick) is unaware of it. It's like a little secret inside the component.

To address this we need to tell react that this component has data(state) that may change overtime.

For this we need a built in function in react called 'useState'

useState

This is considered a 'hook'

A hook is a function that lets us use built in features of react. We will learn about more hooks throughout this course.

With the useState hook we can tell react that we have data that is going to change state.

The state hook returns an array with the first index being a var, and the second being an updater function.

Using 'useState'

The parameter of useState initializes the var, and the param of the updater function updates the var.
