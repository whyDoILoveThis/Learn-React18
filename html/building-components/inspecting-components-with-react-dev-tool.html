Inspecting Components With React Dev Tools

Web Extension

This is the last thing we cover in this section.

Head to the Chrome or Microsoft Edge store and download the React Developer Tools extension.

How to use

Its pretty straight foward and really doesn't need much explanation.

You can see the components, and a component highlighted will highlight the coresponding element in the DOM.

You can search for components.

It will also display information about a component like its props and other info.

If you really want to know more search for on google, but if you are familiar with browser dev tool it should be no problem for you.
