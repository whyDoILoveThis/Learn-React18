Fragments

We can only return a single element

If we try to return multiple elements we get lots of errs.

A simple fix is to wrap it all in a div.

But, this taints our markup with useless elements.

Reacts built-in 'Fragment' components however, doesnt add any elements to our markup.

But we dont want to have to import this everytime and write all that fluff soooooo....

Just the brackets are recomended.
