Creating a React App

2 options..

When creating a new react app we have a choice between the official tool provided by react, or we can use Vite.

Vite is getting increasingly popular because its much faster and produces smaller bundle sizes.

Creating with Vite

In your terminal...

npm create vite@latest

Then...

1. Vite will ask a yes or no question. Press 'y'

2. After that, you will need to name your project.

3. Nex, you select your framework. (of course react this time)

4. Finally, select the language you will use. We are using Typescript in this course.

Final Steps

Vite will tell you to run 3 commands.

Run each of them one at a time.

localhost

You can now use this localhost server just like we always have with live server extension.

It will refresh the page upon save and everything!
