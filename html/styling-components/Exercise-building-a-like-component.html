IMPORTANT TO NOTE

In this exercise we have initialized our state to true, but in a real app this has to be done dynamically based on the content we are displaying to the user.

For example, if we are showing a list of tweets, maybe the user has only liked some of those tweets, not all of them.

So we shouldn't hard code a value like false or true, instead we should have a prop for initializing the status variable.

For this example is doesn't matter though. We'll look at a real example in the project build.
