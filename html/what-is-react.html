What is React?

React is a JS library used for making interactive and dynamic webpages.

It was created by Facebook in 2011, and is the most widely used JS library for frontend development.

Why was React created?

As you know, when a webpage loads the browser takes the html code and cosntructs a tree like structure we call DOM.

This allows us to use JS to change the page content based on user actions.

for example...

const btn = document.querySelector('#btn');

btn.addEventListener('click', () => {

    const div = document.queryselector('#div');

    div.style.display = 'none';

});

Above we have hidden an element upon a button click.

This is 'vanilla' or 'plain' JavaScript.

With React, we dont have to do this anymore.

Instead, we build componants in such a way that, in essence, we create our own html elements that have a complex structure.

Then, instead of mainipulating the DOM, we describe the state of our componants and react manipulates the DOM under the hood

React components help us write reuseable, modular, better organized code.
