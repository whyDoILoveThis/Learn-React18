Connecting to the Backend Intro

Intro

You have seen how we use react to build frontends.

But every application also needs a backend.

You can think of the backend as the engine of an application that powers the frontend.

The backend provides buisness logic, data management, and security features such as authentication and authorization.

There are various languages and tools to build backends.

  • Express.js
  • Django
  • Ruby on Rails
  • Spring
  • ASP.NET Core

Building backends is an entirely different topic and is outside the scope of this course.

But as a react developer you need to know how to connect your apps to the backend.
