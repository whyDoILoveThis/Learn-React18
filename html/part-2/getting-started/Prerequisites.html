Prerequisites

What you should know

The first part of this course was for complete beginners, and convered the basics of react and Typescript.

In the first part we built a web app for discovering video games.

In this part we will add more functionality to that app, so it's ideal if you took the first part.

If you didn't watch the first part then you should have a solid understanding of the following concepts....

  • Basics of Typescript
  • Building Components
  • Styling Components
  • Managing Component State
  • Calling Backend Services

If you are not confident with any of these concepts, I highly encourage studying the first part before taking this one.
