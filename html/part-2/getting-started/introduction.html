What we'll learn

In the first part we learned the fundamental concepts of building web applications with React and Typescript.

In the second part here we will dive a little deeper and learn some advanced concepts such as....

  • Managing Data with React Query
  • State Management with Zustand
  • Routing with React Router

What we'll do

Throughout the course we will enhance the game hub project that we built in the first part.

We'll add chaching to dramatically improve the performance of our application.

We'll also add infinite scrolling so that new games load as we scroll down.

We will ALSO add a page for viewing details about a game

By the end of this course, you will have a solid understanding of how to build robust and scalable web apps with react and Typescript
