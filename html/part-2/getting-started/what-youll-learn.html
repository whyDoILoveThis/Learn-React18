React Query

In the first section we dive into React Query.

React Query is a popular library for data management, and caching in React applications.

With React Query, you never have to use the effect hook to fetch data, take care of errs, and loading indicators.

All those concepts are built into the library.

We also get lots of other features like...

  • Caching
  • Automatic Retry
  • Automatic Refresh
  • Paginated Queries
  • Infinite Queries

By the end of this section we will add caching and infinite scrolling to our game hub project.
