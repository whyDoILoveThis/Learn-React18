Keeping Components Pure

Purity

This is a fundamental concept in React.

In computer science a pure function is a function that given the same input, it always returns the same result.

React was built around this concept. A component should always output the same result if we give it the same props.

So if the props of a component haven't changed, React can skip re-rendering that component.
