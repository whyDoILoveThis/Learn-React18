Updating Arrays

Array example

Let's say we have an array of tags.

When our button is clicked we want to add an item to the array.

Don't push to existing array

The same concept that applys to objects, also applys to arrays.

So we should not mutate or change them, we should provide a brand new array.

That means we dont want to call tags.push() because this modifies the array.

Instead, we should
1. Call setTags
2. Pass it a new array using the spread operator on tags(...tags)
3. Then add any new tags.......

Instead, add to a new array

Removing an object

Maybe we want to remove one of the items from an array

In JS all arrays have a filter method.

Use the filter method to pass an arrow function saying that we want all the tags except the happy tag...

Updating existing array items
