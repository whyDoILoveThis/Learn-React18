Creating a ListGroup Component

Installing Bootstrap

Bootstrap is super easy, fun, productive, and modern-looking. For these reasons we are going to use it in this course.

In you project terminal say...

npm i bootstrap@5.3.2

See the docs for the latest version.

Import Bootstrap

1. You will see an 'index.css' file. It was generated by Vite. Delete it.

2. The code in 'App.css' was also generated by Vite. Clear out everything in this file.

3. Go to 'main.tsx' and change where it imports './index.css' to 'bootstrap/dist/css/bootstrap.css'

Create a Components Folder

Create a new file. 'ListGroup.tsx'

The component

Just do this for now...

function ListGroup () {

  

return <h1>List Group<h1>

}

export default ListGroup;

Now we need to import it to our App component.

CTRL + P in vs code will let you quickly search for a file.

App.tsx

We dont need the 'Message componeent so we can remove that import and replace it with...

import ListGroup from "./components/ListGroup";

Adding a List Group

Go to the Bootstrap Docs and in the Components section you will find a List Group.

Copy their html markup and head back to 'ListGroup.tsx'.

Paste in the makrup.

className

'class' is not allowed so you errs.

Use 'className' instead.
